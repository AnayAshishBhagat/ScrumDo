What is ScrumDo?

ScrumDo is a web app to manage your agile Scrum based projects. We give you the tools to create and track user stories in your project. Watch our intro video to get a taste of what it can do.

What is ScrumDo.org?

The ScrumDo software is open source, and ScrumDo.org is where we keep all the information about that open source project.

What is ScrumDo.com?

The core developers of ScrumDo run this site so you don't have to install and run your own server. It's free for most people!
